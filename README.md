# Econophysics

GBM Visual Original.py  creates a 3d plot with 5 generated Geom Brown Motion and their log-Normal distribution on the z axis
Stock Simulation.py creates a 2d plot with historical and predicted data of a given stock ('AAPL' ticket is default)
Stock Simulation with sigma.py adds the colored area between mu +- sigma

Volatility Smile.ipynb is a Jupyter format program to show how to download Options data and plot Strike Price vs Implied Volatility for the last puts/calls contracts ('AAPL' is taken as default stock)
  (filtering for number of potision held per contract)

Volatility Smile Program.ipynb is a cleaner version of Volatility Smile.ipynb
